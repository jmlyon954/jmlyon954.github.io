# Vendor Insurance Compliance Platform

A lightweight web app prototype for property management teams to manage vendor COI workflows.

## Features

- Vendor intake and risk scoring by service type.
- Insurance requirement mapping for low/medium/high-risk vendors.
- Auto-generated COI sample.
- Initial COI request email template.
- COI analysis against minimum insurance requirements.
- Compliance/deficiency email template.
- Dashboard view with vendor status, expiration, and deficiencies.

## Run locally

Because this app is static HTML/CSS/JS, you can run it with any static server.

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
